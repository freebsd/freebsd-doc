摘要

欢迎使用 FreeBSD！ 本手册适用于安装最新 FreeBSD RELEASE 以及日常使用。手册由 FreeBSD 文档计划持续维护。

本手册中文版在 2025-2026 年期间得到了 FreeBSD-Ask项目（https://github.com/FreeBSD-Ask/Handbook）的积极贡献，特别是 ZFS、Wayland、Wine 等章节的更新。详细致谢见 crossref:preface[preface-acknowledgements, 致谢] 部分。

本手册内容需要不断更新。如有兴趣参与，请联系 {freebsd-doc}。最新英文版见 https://docs.FreeBSD.org/en/books/handbook/ 。您也可以从 FreeBSD FTP 服务器及其镜像站点获取各种格式的文档。
