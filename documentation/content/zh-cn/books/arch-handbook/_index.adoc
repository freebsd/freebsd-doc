摘要

欢迎您阅读 FreeBSD系统结构手册。 这本手册还在不断由许多人_继续书写_。 许多章节还是空白，有的章节亟待更新。 如果您对这个项目感兴趣并愿意有所贡献，请发信给 {freebsd-doc}。
