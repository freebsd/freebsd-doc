= FreeBSD 社群

有超過一百個link:https://www.FreeBSD.org/community/mailinglists/[郵件論壇]、數十個網頁版link:https://forums.FreeBSD.org[論壇]和數個 link:{handbook}#eresources-news[newsgroups] 。 有超過 {{< get-usergroups-info "usergroups" >}} 個link:https://www.freebsd.org/usergroups/[愛用者組織]分佈於全世界 {{< get-usergroups-info "countries" >}} 個國家。也有一個活躍的 link:../community/irc/[IRC] 社群。許多開發者也有建立關於他們 FreeBSD 開發工作的 link:planet.FreeBSD.org[部落格] 開發者和主要貢獻者亦共同維護一個關於 FreeBSD 開發和相關計畫資訊的 link:https://wiki.freebsd.org/[維基網站]。

去年有 {{< get-event-last-year-info "events" >}} FreeBSD 活動在全世界 {{< get-event-last-year-info "countries" >}} 個不同國家舉辦 。近期 FreeBSD 相關活動的 行事曆 和 link:https://www.FreeBSD.org/events/feed.xml[RSS feed] 可以查看 link:https://www.FreeBSD.org/events/[近期活動] 頁面。有許多過往活動的影片 在 Youtube 的 link:https://www.youtube.com/bsdconferences[BSD Conferences] 頻道。

== 社群網站

FreeBSD 亦出現於許多不同的社群網站。

* https://www.flickr.com[flickr] 有數千張愛用者組織、會議和黑客松的照片標記 'link:https://flickr.com/search/?z=t&ss=2&w=all&q=freebsd&m=text[freebsd]'
* https://www.youtube.com[YouTube] 有數百個 https://www.youtube.com/results?search_query=freebsd&search_type=&aq=f[FreeBSD] 相關的會議、螢幕錄影或是範影片。特別是有一個新的 https://www.youtube.com/bsdconferences[BSD Conferences] 頻道有超過一個小時 FreeBSD 會議的演講影片。
* 有一個 https://www.facebook.com/groups/FreeSBD/[FreeBSD Users Group] 在 https://www.facebook.com[Facebook] 和一個 https://www.linkedin.com/groups?gid=47628[FreeBSD Group] 在 https://www.linkedin.com[LinkedIn]。
* 你可以在 https://twitter.com[Twitter] 追蹤 https://twitter.com/freebsd[@freebsd], https://twitter.com/freebsdhelp[@freebsdhelp], https://twitter.com/freebsdblogs[@freebsdblogs], 或是 https://twitter.com/freebsdcore[@freebsdcore] 。
