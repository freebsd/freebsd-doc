= Старт с FreeBSD

Операционная система FreeBSD доступна в виде образов, опубликованных на этой странице, а также напрямую у поставщиков облачных решений.
Перед тем, как скачивать тот или иной образ, пожалуйста, прочтите link:{handbook}bsdinstall/[**руководство по установке**].

Для получения информации о поддерживаемых архитектурах обратитесь к разделу о link:../platforms/#_supported_platforms[платформах].

Для получения общей информации о прошлых, актуальных и будущих релизах обратитесь к link:../releases/[Информации о релизах]
или http://ftp-archive.FreeBSD.org/pub/FreeBSD-Archive/old-releases/[Историческому архиву релизов].

== Выбор образа

Для поддержки различных сценариев установки и разворачивания образы FreeBSD публикуются в нескольких форматах:

* Установочные образы: образы ISO дисков DVD (`disc1`, `dvd1`) позволяют осуществлять установку без подключения к сети; образ ISO `bootonly` требует сетевого подключения. Образы накопителей USB доступны в полном варианте `memstick` (без сетевого подключения) и в варианте `mini-memstick` (требуется наличие сети).
* Образы виртуальных машин: Предустановленные системы для гипервизоров и облачных сред. Образы `BASIC-CI` представляют собой минимальные сборки FreeBSD с установленным по умолчанию совместимым с cloudinit механизмом работы со скриптами man:nuageinit[7], что позволяет гибко управлять сценариями запуска системы. Такие скрипты используются в других проектах для запуска сценариев тестирования; они не являются частью собственной инфраструктуры CI проекта FreeBSD.
* Образы карт памяти SD: Предустановленные системы для одноплатных вычислительных систем, таких как Raspberry Pi.

== Промежуточные релизы разработки

link:../snapshots/[Образы промежуточных релизов] публикуются в целях тестирования экспериментальных функций и выявления дефектов.

* Образы `CURRENT` являются еженедельными промежуточными релизами, формируемыми для тестирования самой актуальной основной ветки разработки.
* Образы `STABLE` являются еженедельными промежуточными релизами, формируемыми для тестирования того, что станет предрелизными образами.
* `ALPHA`, `BETA` и кандидаты на релизную версию (`RC`) являются предрелизными образами. Пожалуйста, тестируйте их при их наличии.

Перед тем, как скачать один из этих образов, пожалуйста, прочтите link:{handbook}cutting-edge/#current-stable[**руководство по отслеживанию разработки**].
