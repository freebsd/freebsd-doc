= Подключаемые модули OpenSearch для FreeBSD

В правом верхнем углу Firefox находится Панель поиска, в которой Firefox размещает несколько популярных поисковых сайтов. Щёлкните на ней, чтобы добавить _Поисковые сервисы FreeBSD_ в Панель поиска Firefox.
