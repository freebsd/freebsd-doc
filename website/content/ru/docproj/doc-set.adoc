= FreeBSD Documentation Project: Documentation Set

Документация по FreeBSD разбивается на три основных категории:

. https://man.freebsd.org/cgi/man.cgi[*Страницы Справочника*]
+
Страницы справочника являются частью базовой операционной системы. И проект документации, и коммиттеры исходного кода изменяют и следят за актуальность страниц Справочника.
+
Японская команда перевела многие страницы Справочника на японский язык. Нет причин, по которым другие добровольцы не могли бы начать перевод страниц Справочника на другие языки.
. link:https://docs.freebsd.org/ru/books/[*Книги*]
+
В проекте имеется большое количество документации, достигшей "книжной длины", или стремящейся к ней. Сюда включены FreeBSD FAQ и Руководство FreeBSD.
. link:https://docs.freebsd.org/ru/articles[*Статьи*]
+
В FreeBSD есть масса информации, доступной в более коротком формате статьи, аналогично учебникам или HOWTO других проектов.
