= Информация о релизах

[.right]
image:../../gifs/releases.jpg[Релизы FreeBSD,width=300,height=200]

Релизы FreeBSD подразделяются на "продуктивные релизы" и "старые продуктивные релизы".
Первый тип лучше всего подходит для большинства пользователей.
Второй тип для пользователей, желающих придерживаться более консервативной стратегии обновления.
