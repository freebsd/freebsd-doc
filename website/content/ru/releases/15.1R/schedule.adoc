== Введение

Это план подготовки релиза для FreeBSD {localRel}.
Для получения дополнительной информации о процессе подготовки релизов, пожалуйста, ознакомьтесь с разделом link:../../../releng/[Информация о подготовке релизов] на веб-сайте.

Общие обсуждения предстоящего релиза и известных проблем следует направлять в публичный список рассылки mailto:FreeBSD-{localMailingList}@FreeBSD.org[freebsd-{localMailingList}].
