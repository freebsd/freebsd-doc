== Введение

Это план выпуска для FreeBSD {localRel}.
Для получения дополнительной информации о процессе инженерии выпусков, пожалуйста, ознакомьтесь с разделом link:../../../releng/[Информация о подготовке релизов] на веб-сайте.

Общие обсуждения предстоящего выпуска и известных проблем следует направлять в публичный список рассылки mailto:FreeBSD-{localMailingList}@FreeBSD.org[freebsd-{localMailingList}].
