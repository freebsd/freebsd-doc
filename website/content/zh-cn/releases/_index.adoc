= 发行版本信息

[.right]
image:../../gifs/releases.jpg[FreeBSD Releases,width=300,height=200]

FreeBSD 发行版本分为 “生产版本” (Production Releases) 和 “旧式版本” (Legacy Releases)。 前者适用于希望使用最新特性的用户， 后者适合于希望采取更保守的升级策略的用户。

每个发行版本的 HTML 格式的相关文档文件， 都可以在 link:../relnotes/[发行文档] 页面找到。
