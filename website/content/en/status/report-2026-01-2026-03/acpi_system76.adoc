=== ACPI Driver for System76 Laptops

Contact: Pouria Mousavizadeh Tehrani <pouria@FreeBSD.org>

I have been working on a dedicated driver for System76 laptops and it is now available on CURRENT.

So far, I have added the support for:

* Battery charging thresholds. link:https://cgit.freebsd.org/src/commit/?id=f87ba4522ec9e7b2227b8f20f3a4d7c6a129da1c[(commit)]
* Keyboard brightness with man:backlight[8] support. link:https://cgit.freebsd.org/src/commit/?id=105869a2c78d21f310a8f271eaa510acea045805[(commit)]
* Keyboard RGB color handling. link:https://cgit.freebsd.org/src/commit/?id=cdad55809ef59239c3bbdc841ed307db68bb3971[(commit)]

The only thing left is switching between dGPU and iGPU by the driver.
However, it is possible to switch without driver assistance.
