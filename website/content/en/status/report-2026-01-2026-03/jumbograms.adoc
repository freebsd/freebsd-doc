=== Removal of RFC 2675 Support

Contact: Tom Jones <thj@FreeBSD.org>

The Jumbo Payload Option is an IPv6 extension header intended to support packet sizes in excess of 65,735 octets.
FreeBSD gained Jumbo Payload support from the KAME project, but not real networks ever carried support.

As part of modernising the IPv6 stack Jumbo support has been removed.
As far as we can tell it has never been possible to use this support with a FreeBSD host.
