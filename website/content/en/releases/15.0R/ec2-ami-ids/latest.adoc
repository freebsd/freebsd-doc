= FreeBSD 15.0-RELEASE EC2 Patched AMIs

== Introduction

This page lists EC2 AMIs for FreeBSD 15.0-RELEASE with the latest post-release security and errata fixes applied.
You can filter the list of AMIs by region, architecture, flavour, and/or root filesystem.

A separate page lists the link:../release[original images from the release] without any post-release security and errata fixes.

These AMI IDs are available programmatically via the AWS SSM Parameter Store:

....
$ aws --region ${REGION} ssm get-parameter \
    --query 'Parameter.Value' --output text \
    --name /aws/service/freebsd/${ARCH}/${FLAVOUR}/${FILESYSTEM}/15.0/RELEASE/latest
....

These AMI IDs (on this page and also in the SSM Parameter Store) will be updated when new AMIs are built for security and errata updates.
