= FreeBSD 15.1-RELEASE EC2 AMIs

== Introduction

This page lists EC2 AMIs for FreeBSD 15.1-RELEASE.
You can filter the list of AMIs by region, architecture, flavour, and/or root filesystem.

Note that these are the original images from the release and do not have any post-release security or errata fixes; you may want the link:../latest[latest patched AMIs for FreeBSD 15.1-RELEASE] instead.

These AMI IDs are available programmatically via the AWS SSM Parameter Store:

....
$ aws --region ${REGION} ssm get-parameter \
    --query 'Parameter.Value' --output text \
    --name /aws/service/freebsd/${ARCH}/${FLAVOUR}/${FILESYSTEM}/15.1/RELEASE
....
