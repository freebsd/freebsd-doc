== メーリングリストのアーカイブ

https://lists.FreeBSD.org/index/[メーリングリストのアーカイブ] のインデックスは毎週更新しています!

メーリングリストアーカイブを読むもう一つの方法は http://lists.FreeBSD.org/pipermail[Pipermail リストアーカイブ] を利用することです (2003 年 3 月から 2021 年 5 月の期間のメールが保存されています)。

このメーリングリスト (そのほかのも) は https://freebsd.markmail.org/search/?q=[MarkMail] でもアーカイブされています。
